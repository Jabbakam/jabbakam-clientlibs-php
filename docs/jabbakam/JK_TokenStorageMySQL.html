JK_TokenStorageMySQL

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • JK_TokenStorage

Class that stores Jabbakam OAuth tokens in a mySQL database.

Uses (and depends on) the mySQLi library.

  1.  CREATE TABLE `jabbakam_tokens` (
  2.  `type` ENUM( 'request', 'access' ) NOT NULL ,
  3.  `user_id` INT UNSIGNED NOT NULL ,
  4.  `key` VARCHAR( 64 ) NOT NULL ,
  5.  `secret` CHAR( 32 ) NOT NULL ,
  6.  `expires` INT UNSIGNED NOT NULL DEFAULT '0' COMMENT 'UNIX time UTC',
  7.  PRIMARY KEY ( `type` , `user_id` )
  8.  ) ENGINE = MYISAM CHARACTER SET utf8 COLLATE utf8_general_ci COMMENT = 'Holds Jabbakam API OAuth tokens';
  9.  
  10.  ALTER TABLE `jabbakam_tokens` ADD INDEX ( `type` , `key` ) ;
